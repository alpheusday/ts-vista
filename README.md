# TypeScript Vista

A utility to enhance TypeScript.

## Installation

Install this package as a dependency in the project:

```sh
# npm
npm i ts-vista

# Yarn
yarn add ts-vista

# pnpm
pnpm add ts-vista

# Deno
deno add npm:ts-vista

# Bun
bun add ts-vista
```
